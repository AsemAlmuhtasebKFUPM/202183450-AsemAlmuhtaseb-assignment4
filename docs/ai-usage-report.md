# AI Usage Report

## Tools Used & Use Cases

### 1. ChatGPT
I used ChatGPT throughout the assignment as a support tool for problem-solving, explanations, and code improvements. I used it to:
- understand assignment requirements and choose suitable features for the portfolio
- solve coding problems and debug issues in HTML, CSS, and JavaScript
- improve existing features such as project search, filtering, sorting, and API integration
- get explanations for how features work, such as `localStorage`, form handling, filtering logic, and API fetching
- refine documentation and project structure

I did not use ChatGPT output blindly. I reviewed every suggestion, edited the code myself, and adjusted it to fit my project structure and assignment needs.

### 2. GitHub Copilot
I used GitHub Copilot mainly as a coding assistant inside the editor. I used it for:
- code completion while writing HTML, CSS, and JavaScript
- helping with indentation correctness and formatting consistency
- speeding up repetitive parts of coding
- suggesting small syntax fixes while I was editing files

I still checked Copilot suggestions before keeping them and made manual edits when needed.

## Benefits & Challenges

### Benefits
Using AI helped me in several ways:
- it made problem-solving faster when I got stuck
- it helped me understand programming concepts more clearly
- it improved my workflow by giving quick suggestions and explanations
- it helped me organize features more effectively
- it reduced small coding mistakes such as syntax and indentation issues

### Challenges
There were also some limitations and difficulties:
- some AI suggestions needed adjustment before they worked correctly in my project
- not every API suggestion worked immediately, so I had to test and replace some solutions
- AI-generated code sometimes needed changes to match my exact file structure and design
- I had to carefully review suggestions to avoid adding things I did not fully understand

## Learning Outcomes

Using AI in this assignment helped me learn:
- how to implement interactive JavaScript features such as filtering, sorting, and theme persistence
- how to work with `localStorage` to save user preferences
- how to connect to an external API and handle errors properly
- how to improve frontend structure using cleaner HTML, CSS, and JavaScript
- how to debug step by step instead of guessing
- how AI can support development without replacing my own understanding and decisions

It also improved my workflow by helping me move faster between coding, debugging, testing, and refining the final result.

## Responsible Use & Modifications

I used AI responsibly as a support tool, not as a replacement for my own work. I reviewed all AI suggestions before applying them and made sure I understood what they were doing. I edited every AI-generated suggestion to fit my own portfolio, assignment requirements, naming, structure, and design.

To maintain correctness, originality, and academic integrity:
- I tested the code after making changes
- I modified AI suggestions instead of copying everything exactly as given
- I kept only the parts that fit my project requirements
- I made manual improvements and corrections where needed
- I ensured that the final submission reflected my own understanding and editing

In particular, I used ChatGPT for explanations, debugging, and feature guidance, and I used GitHub Copilot mainly for code completion and indentation correctness while writing and editing the project files.
